About me

Saya adalah siswi SMKN 2 Kota Bekasi yang sedang belajar di bidang desain, khususnya desain grafis, UI/UX, dan dasar web design. Meski masih dalam tahap belajar, saya memiliki semangat tinggi untuk mengembangkan keterampilan, terbuka terhadap masukan, serta ingin mendapatkan pengalaman magang untuk belajar langsung dari dunia kerja kreatif.

Selama proses belajar, saya sudah mulai terbiasa menggunakan beberapa software desain seperti Photoshop, dan Figma untuk membuat desain grafis serta tampilan antarmuka sederhana. Saya juga mempelajari dasar HTML dan CSS agar lebih memahami hubungan antara desain dan implementasi di web. Walaupun masih pemula, saya senang bereksperimen dengan berbagai gaya desain dan terus mencari inspirasi dari sumber-sumber kreatif.
